# Шаблон: Эскалация на менеджера

## Когда использовать
Ситуация выходит за рамки стандартных — возврат, блокировка провайдером, опт, конфликт

## Шаблон для оператора (что написать клиенту)

### При запросе возврата
Понял вас. Передаю вопрос менеджеру — он свяжется с вами в ближайшее время.

### При блокировке провайдером
Принято, занимаемся вашим вопросом. Дадим обратную связь в течение {{срок}}.

### При оптовом запросе
Мы заинтересованы в сотрудничестве! Условия зависят от объёма — передаю менеджеру для обсуждения деталей.

### При конфликте / агрессии
Понимаем вашу ситуацию. Передаю старшему менеджеру для оперативного решения.

## Пояснение для оператора
- Зафиксируй суть проблемы
- Передай менеджеру с контекстом
- Не обещай конкретных сроков, если не согласовано

## Переменные
- `{{срок}}` — "24 часов" / "сегодня" / "в ближайшее время"
