# Шаблон: Проверка статуса / подтверждение

## Когда использовать
После выдачи товара, проверка что клиент всё получил

## Шаблоны (от коротких к длинным)

### Быстрая проверка
```
Удалось?
```

### Подтверждение приёмки
```
Принято?
Всё работает?
```

### Ожидание обратной связи
```
Ждём обратную связь.
Отпишитесь по готовности.
```

### Подтверждение от оператора
```
Принято
Платеж получен
Взяли в работу
```

### Напоминание
```
Добрый день/вечер!
Напоминаем о себе.
Есть возможность проверить?
```

## Примечания
- «Удалось?» — самый частый вопрос оператора (45+ раз в базе)
- Использовать после каждой выдачи учётных данных
- Если клиент не отвечает 24ч — напоминание
