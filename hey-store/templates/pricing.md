# Шаблон: Расчёт стоимости

## Когда использовать
Клиент заказывает несколько товаров, нужен расчёт

## Шаблон

{{перечень_товаров}}
———
К оплате: {{итого}} USDT

## Формат перечня

```
{{количество}} × {{название}} — {{цена}} USDT
{{количество}} × {{название}} — {{цена}} USDT
Итого: {{сумма}} USDT
```

## Примеры из реальных диалогов

```
3 esim — 90
3 bybit + proxy — 105
Итого: 195 USDT
```

```
35 × 3 + 30 × 3 = 195 USDT
```

## Базовые цены (уточнять актуальные!)
- eSIM: ~30 USDT
- Bybit аккаунт: ~35 USDT
- Wise (на другое имя): ~70 USDT
- Wise (на своё имя): ~120 USDT
- RedotPay: ~35 USDT
- Payoneer: ~100 USDT

## Переменные
- `{{перечень_товаров}}` — список позиций с ценами
- `{{итого}}` — общая сумма
