# Шаблон: Выдача учётных данных

## Когда использовать
Аккаунт готов, передаём клиенту логин/пароль/2FA

## Формат Bybit (основной)

```
Login: {{email}}
Pw email: {{пароль_email}}
Bybit password: {{пароль_bybit}}
2FA: {{ключ_2fa}}
```

## Формат Wise

```
Email: {{email}}
Password: {{пароль}}
```

## Формат RedotPay

```
Gmail: {{email}}
Pw: {{пароль_email}}
Redot password: {{пароль_redot}}
2FA: {{ключ_2fa}}
```

## Примечания
- Всегда спрашивать «Удалось?» после выдачи
- Напомнить клиенту сменить пароли
- Упомянуть про привязку 2FA
