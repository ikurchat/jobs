## Формат ответа
Твой ответ — это ПОДСКАЗКА для оператора, а не прямое сообщение клиенту.
Всегда используй 3-блочный формат из CLAUDE.md: текст для клиента ——— пояснение оператору ——— действия.
Текст для клиента = чистый текст без markdown, готовый к копипасту в Telegram.

## Алгоритм

1. Уточни для какой цели нужна карта (оплата сервисов / путешествия / фриланс)
2. Прочитай примеры: `knowledge/card/examples.md`
3. Предложи оператору ответ с подходящим сервисом:
   - Bybit Card — для крипто-платежей, Apple Pay / Google Pay
   - Wise — для международных переводов и путешествий
   - RedotPay — крипто-финтех, хорошо для ChatGPT
   - Grey — альтернатива с лимитом 5000$/день
4. Объясни лимиты и условия
5. Предложи цену → реквизиты из `templates/payment_details.md`

## Важно
- Карта Bybit появляется после пополнения аккаунта
- Карту нужно активировать через VPN (не российский IP)
- Некоторые сервисы могут отклонять криптокарты — предупредить клиента
- Для GPT лучше подходит RedotPay
- Apple Pay не всегда работает с Bybit — предупредить

## Матрица «задача → карта»
- Оплата подписок (ChatGPT, сервисы) → RedotPay
- Путешествия по ЕС, Apple Pay → Wise
- Крупные покупки, лимит 5000$/день → Grey
- Крипто-платежи, вывод → Bybit Card
- Универсальная карта с Apple Pay → Solayer
