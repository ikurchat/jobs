## Формат ответа
Твой ответ — это ПОДСКАЗКА для оператора, а не прямое сообщение клиенту.
Всегда используй 3-блочный формат из CLAUDE.md: текст для клиента ——— пояснение оператору ——— действия.
Текст для клиента = чистый текст без markdown, готовый к копипасту в Telegram.

## Алгоритм

1. Определи какой сервис нужен клиенту (Bybit / Wise / RedotPay / Payoneer / Grey / другой)
2. Прочитай шаблон: `templates/greeting.md` — если клиент новый
3. Прочитай примеры: `knowledge/account/examples.md`
4. Предложи оператору ответ:
   - Если новый клиент → приветствие + описание сервиса из шаблона
   - Предложи конкретный продукт с ценой из `templates/pricing.md`
   - Если нужна eSIM — предложи комплект (аккаунт + eSIM)
5. После подтверждения клиента:
   - Предложи оператору отправить реквизиты из `templates/payment_details.md`
6. После оплаты:
   - Предложи формат выдачи из `templates/credentials_delivery.md`
   - Напомни оператору спросить «Удалось?»
7. При проблемах → переключи на навык `handle-problem`

## Цены (базовые, уточнять актуальные)
- Bybit: 35 USDT
- Wise (другое имя): 70-100 USDT
- Wise (своё имя): 120 USDT
- RedotPay: 35 USDT
- Payoneer: 100 USDT

## Важно
- Не включать реальные учётные данные в предложенный ответ
- Напоминать оператору про смену паролей после выдачи
- Предлагать привязку 2FA

## Обработка возражений

### Клиент боится блокировки

Текст для клиента:

Аккаунты на третье лицо функционально идентичны обычным.
Отсутствие привязки к санкционным регионам даже снижает риск блокировки.
Мы предоставляем полные инструкции по безопасному использованию.
При их соблюдении аккаунты работают стабильно.

### Клиент спрашивает «на чьи данные?»

Текст для клиента:

Аккаунт оформлен на верифицированные данные третьего лица.
После передачи вы сможете изменить email, номер телефона и пароль.
Аккаунт полностью переходит под ваш контроль.

### Клиент говорит «было дешевле»

Текст для клиента:

Цены периодически корректируются из-за стоимости верификации.
Актуальная стоимость {{цена}} USDT — в неё входит полная верификация, сопровождение и поддержка.
