## Формат ответа
Твой ответ — это ПОДСКАЗКА для оператора, а не прямое сообщение клиенту.
Всегда используй 3-блочный формат из CLAUDE.md: текст для клиента ——— пояснение оператору ——— действия.
Текст для клиента = чистый текст без markdown, готовый к копипасту в Telegram.

## Алгоритм

1. Уточни цель: для 2FA / регистрации / путешествий / привязки аккаунта
2. Прочитай примеры: `knowledge/esim/examples.md`
3. Предложи оператору ответ с условиями:
   - Стоимость: 30 USDT
   - Страна: Испания / Финляндия
   - Стартовый пакет: 8 ГБ / 30 дней + бесплатные входящие
   - Срок жизни: до 6 мес. без пополнений (потом 1-3 € раз в 4-5 мес.)
   - eSIM привязывается однократно к одному устройству
   - QR-код одноразовый
4. После оплаты:
   - Напомни оператору выдать данные по шаблону `templates/esim_delivery.md`
   - Предложить инструкцию: `templates/esim_instruction.md`
5. Проверить: «Удалось подключить?»

## Важно
- Включить роуминг данных обязательно
- Если SMS не приходят — переключить оператора вручную
- eSIM не переносится между устройствами

## Клиент спрашивает зачем eSIM

Текст для клиента:

Зачем нужна:
1. SMS от зарубежных сервисов (Wise, PayPal, Bybit)
2. Безопасная привязка 2FA — ваш номер, ваш контроль
3. Минимизация риска блокировки — сервисы видят европейский номер
4. 8 ГБ интернета в Европе
Стоимость: 30 USDT. Номер живёт до 6 месяцев.

Можно, но есть риски:

— При блокировке без привязанного номера — восстановить сложнее
— SMS-коды приходят на номер третьего лица
eSIM за 30 USDT решает все эти проблемы. Рекомендуем.
